Internet es una enorme red formada por dispositivos, cables, servidores, routers y protocolos que permiten intercambiar información a escala mundial.

Cuando visitas una página web, varios sistemas trabajan juntos en pocos segundos para localizar el sitio, solicitar sus archivos y mostrarlos en el navegador.

¿Qué es Internet?

Internet es una red mundial de redes informáticas conectadas entre sí.

Permite que computadoras, teléfonos, servidores y otros dispositivos compartan información utilizando reglas de comunicación comunes.

Ejemplo sencillo

Cuando envías un mensaje, tu dispositivo transmite datos por una red hasta llegar al servicio que los entrega al destinatario.

Internet permite utilizar servicios como:

  • Páginas y aplicaciones web.
  • Correo electrónico.
  • Mensajería instantánea.
  • Videollamadas.
  • Almacenamiento en la nube.
  • Transmisión de audio y video.
  • Juegos en línea.
  • Servicios financieros.

¿Qué es una red informática?

Una red informática es un conjunto de dispositivos conectados para intercambiar datos y compartir recursos.

Una red puede incluir:

Computadoras

Envían, reciben y procesan información.

Teléfonos

Acceden a servicios mediante Wi-Fi o datos móviles.

Impresoras

Pueden compartirse entre varios usuarios.

Servidores

Ofrecen páginas, archivos, aplicaciones y otros servicios.

Amplía este tema en: ¿Qué es una red informática?

Tipos de redes

LAN

Red local que conecta dispositivos dentro de una casa, oficina o edificio.

WLAN

Red local inalámbrica que normalmente utiliza Wi-Fi.

WAN

Red extensa que conecta ubicaciones separadas por grandes distancias.

PAN

Red personal de corto alcance, como una conexión Bluetooth.

Internet es una red de redes

Conecta muchas redes locales y extensas mediante proveedores, cables terrestres, fibra óptica y otros sistemas de comunicación.

Dispositivos principales de una red

Router

El router dirige la información entre diferentes redes. En una casa, normalmente conecta la red local con el proveedor de Internet.

Switch

Un switch conecta varios dispositivos dentro de una misma red local y dirige los datos hacia el destino correspondiente.

Módem

El módem establece la comunicación con la tecnología utilizada por el proveedor de Internet.

Punto de acceso

Permite que dispositivos inalámbricos se conecten a una red mediante Wi-Fi.

En equipos domésticos

Un mismo aparato puede integrar módem, router, switch y punto de acceso inalámbrico.

¿Qué es una dirección IP?

Una dirección IP identifica un dispositivo o interfaz dentro de una red.

Ejemplo de IPv4 
192.168.1.25
Ejemplo simplificado de IPv6 
2001:db8::25

Dirección pública

Identifica una conexión o servicio frente a otras redes de Internet.

Dirección privada

Se utiliza dentro de redes locales y normalmente no es accesible directamente desde Internet.

Las direcciones pueden cambiar

Muchos dispositivos reciben una dirección automáticamente durante un tiempo determinado.

¿Qué es el DNS?

El DNS permite utilizar nombres fáciles de recordar en lugar de escribir direcciones IP.

Nombre de dominio 
bytenova.app

Cuando escribes un dominio, el sistema DNS busca la dirección necesaria para localizar el servicio.

1

Escribes el dominio

El navegador recibe el nombre del sitio.

2

Se consulta el DNS

El sistema busca la dirección relacionada.

3

Se obtiene una dirección

El navegador puede localizar el servidor.

¿Qué es TCP/IP?

TCP/IP es un conjunto de protocolos utilizado para organizar la comunicación entre dispositivos.

IP

Ayuda a identificar el origen, el destino y la ruta de los paquetes.

TCP

Ayuda a entregar los datos correctamente y en el orden esperado.

La información puede dividirse en pequeñas unidades llamadas paquetes.

Cada paquete puede viajar por diferentes rutas antes de llegar al destino y ser reconstruido.

Continúa con: ¿Qué es TCP/IP?

Clientes y servidores

En muchos servicios de Internet existe una relación entre un cliente y un servidor.

Cliente

Solicita información o una función. Un navegador puede actuar como cliente.

Servidor

Recibe la solicitud, la procesa y devuelve una respuesta.

Ejemplo

El navegador solicita una página y el servidor responde enviando los archivos necesarios.

¿Qué es una URL?

Una URL es la dirección utilizada para localizar un recurso dentro de la web. 

https://bytenova.app/pages/articulos.html

Partes principales

Protocolo

En el ejemplo es https.

Dominio

En el ejemplo es bytenova.app.

Ruta

Indica el recurso solicitado dentro del sitio.

Aprende más en: ¿Qué es una URL?

HTTP y HTTPS

HTTP es un protocolo utilizado para intercambiar recursos entre navegadores y servidores web.

HTTPS agrega protección mediante cifrado durante la comunicación.

HTTP

Transfiere solicitudes y respuestas web.

HTTPS

Protege la comunicación frente a observación y modificaciones durante el tránsito.

HTTPS no garantiza que el sitio sea confiable

Una página fraudulenta también puede utilizar una conexión cifrada. Siempre debes revisar el dominio y el contenido.

Cómo llega una página hasta el navegador

1

Escribes una dirección

El navegador interpreta la URL.

2

Se consulta el DNS

Se busca la dirección asociada al dominio.

3

Se establece la conexión

El cliente se comunica con el servidor.

4

Se envía una solicitud

El navegador solicita el recurso mediante HTTP o HTTPS.

5

El servidor responde

Envía HTML, CSS, JavaScript, imágenes u otros archivos.

6

El navegador construye la página

Interpreta los archivos y muestra el resultado.

¿Cómo funciona una red Wi-Fi?

Wi-Fi permite conectar dispositivos de forma inalámbrica dentro del alcance de un punto de acceso.

El dispositivo envía y recibe datos mediante ondas de radio en lugar de utilizar un cable directo.

Factores que afectan la conexión

  • Distancia respecto al punto de acceso.
  • Paredes y obstáculos.
  • Interferencia de otros equipos.
  • Cantidad de dispositivos conectados.
  • Capacidad del router.
  • Velocidad contratada con el proveedor.
  • Estado del dispositivo.
Wi-Fi y velocidad de Internet no son lo mismo

Puedes tener un buen servicio contratado, pero una señal inalámbrica débil dentro de la casa.

Seguridad básica de una red

  • Cambia la contraseña predeterminada del router.
  • Utiliza una contraseña Wi-Fi larga y única.
  • Mantén actualizado el firmware.
  • Utiliza un cifrado moderno disponible.
  • Desactiva funciones que no necesites.
  • No compartas la contraseña públicamente.
  • Revisa los dispositivos conectados.
  • Crea una red separada para invitados.

Firewall

Un firewall controla conexiones según reglas de seguridad.

Consulta: ¿Qué es un firewall?

VPN

Una VPN puede crear una conexión cifrada entre el dispositivo y un servicio remoto.

No evita contraseñas débiles, malware ni páginas falsas.

Consulta: ¿Qué es una VPN?

Diagnóstico básico de una conexión

Cuando Internet no funciona, comprueba el problema de forma ordenada.

1

Revisa la conexión física

Comprueba cables, luces y alimentación de los equipos.

2

Prueba otro dispositivo

Ayuda a determinar si el problema es general o solamente afecta a un equipo.

3

Reinicia la conexión

Desconecta y vuelve a conectar el Wi-Fi o el cable.

4

Comprueba otros sitios

El problema podría afectar solamente a un servicio.

5

Reinicia el router

Espera unos segundos antes de volver a encenderlo.

6

Contacta al proveedor

Hazlo cuando la conexión general continúe sin funcionar.

Comandos básicos de diagnóstico

Consultar configuración en Windows 
ipconfig
Consultar interfaces en Linux 
ip address
Comprobar comunicación 
ping bytenova.app
Un resultado fallido no identifica siempre la causa

Algunos servidores bloquean ciertas pruebas y una falla puede relacionarse con DNS, el proveedor, el dispositivo o la red local.

Conclusión

Internet funciona gracias a redes, dispositivos, direcciones y protocolos que permiten localizar y transportar información.

Comprender conceptos como IP, DNS, TCP/IP, servidores, URL y HTTPS ayuda a utilizar la tecnología con mayor seguridad y resolver problemas básicos de conexión.
