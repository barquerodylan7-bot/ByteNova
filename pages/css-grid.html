CSS Grid: crea diseños web en filas y columnas

Publicado por ByteNova · 22 de julio de 2026

En este artículo aprenderás:

Descubre cómo crear cuadrículas adaptables con filas, columnas, áreas y espacios consistentes.

Muchos diseños necesitan controlar filas y columnas al mismo tiempo. Grid ofrece una cuadrícula clara y flexible.

¿Qué es?

CSS Grid es un sistema bidimensional que distribuye elementos en filas y columnas definidas por un contenedor.

Idea clave

Descubre cómo crear cuadrículas adaptables con filas, columnas, áreas y espacios consistentes.

¿Por qué es importante?

Es útil para galerías, paneles, páginas completas y listados de tarjetas con tamaños adaptables.

Comprender este fundamento permite tomar mejores decisiones, reconocer límites y relacionar la teoría con situaciones reales. La práctica debe centrarse en problemas pequeños antes de aumentar la complejidad.

Contenedor grid

Se activa con display:grid.

fr

gap

minmax

Tamaño mínimo y máximo.

Áreas

Regiones con nombre.

Tarjetas adaptables

.rejilla {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(240px, 1fr));
  gap: 1.5rem;
}

La cantidad de columnas cambia según el ancho.

Cuadrículas explícitas, implícitas y adaptables

CSS Grid trabaja con filas y columnas al mismo tiempo. Las columnas definidas con grid-template-columns forman la cuadrícula explícita; las filas o columnas creadas automáticamente para elementos adicionales forman la cuadrícula implícita. Propiedades como grid-auto-rows permiten controlar ese comportamiento.

Las unidades fraccionarias reparten el espacio disponible. La combinación repeat(auto-fit, minmax(240px, 1fr)) crea tantas columnas como quepan, respeta un tamaño mínimo y distribuye el espacio sobrante. Es una herramienta poderosa, pero el tamaño mínimo debe probarse con contenido real, no elegirse al azar.

.galeria {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(240px, 1fr));
  gap: 1.25rem;
}

.destacada {
  grid-column: span 2;
}

Práctica guiada: panel de contenidos

Diseña un panel con una tarjeta destacada, cuatro tarjetas normales y una barra lateral. En computadora utiliza áreas de cuadrícula; debajo de un ancho razonable, pasa a una sola columna. Comprueba que el orden visual coincida con el orden del HTML para no perjudicar la navegación con teclado.

  • Empieza con el contenido sin estilos.
  • Define primero las columnas generales.
  • Usa gap para separación consistente.
  • Evita colocar elementos visualmente en un orden distinto al del documento.

Cómo empezar a practicar

  1. Crea tres columnas.
  2. Agrega gap.
  3. Usa fr.
  4. Prueba auto-fit.
  5. Crea barra lateral.

Conserva cada ejercicio y escribe una breve nota sobre la decisión que tomaste. Después crea una segunda versión más clara o más resistente a errores.

Preguntas frecuentes

¿Es responsive?

Puede serlo con unidades flexibles.

¿Qué es 1fr?

Una parte proporcional del espacio libre.

¿auto-fit y auto-fill?

Manejan distinto las columnas vacías.

¿Grid y Flexbox juntos?

Sí.

¿Puede cambiar el orden?

Sí visualmente, pero conviene conservar el orden lógico.
