CSS Flexbox: guía para alinear y distribuir elementos

Publicado por ByteNova · 22 de julio de 2026

En este artículo aprenderás:

Aprende cómo Flexbox alinea y distribuye elementos en filas o columnas adaptables.

Flexbox simplifica la alineación de menús, botones y tarjetas a lo largo de un eje principal.

¿Qué es?

Es un sistema de diseño unidimensional de CSS donde un contenedor controla dirección, alineación, separación y tamaño de sus hijos.

Idea clave

Aprende cómo Flexbox alinea y distribuye elementos en filas o columnas adaptables.

¿Por qué es importante?

Reduce reglas frágiles y se adapta a diferentes anchos, por lo que resulta ideal para componentes de interfaz.

Comprender este fundamento permite tomar mejores decisiones, reconocer límites y relacionar la teoría con situaciones reales. La práctica debe centrarse en problemas pequeños antes de aumentar la complejidad.

Conceptos principales

Contenedor

Se activa con display:flex.

Eje principal

Depende de flex-direction.

Eje transversal

Es perpendicular al principal.

justify-content

Distribuye en el eje principal.

align-items

Alinea en el transversal.

gap

Crea separación uniforme.

Cómo aplicarlo paso a paso

1

Selecciona

Aplica flex al contenedor.

2

Define dirección

row o column.

3

Alinea

Configura justify-content y align-items.

4

Permite salto

Usa flex-wrap.

5

Ajusta tamaño

Controla crecimiento y base.

Barra flexible

.navegacion {
  display: flex;
  align-items: center;
  justify-content: space-between;
  gap: 1rem;
  flex-wrap: wrap;
}

Puede distribuir elementos y saltar de línea.

¿Dónde se utiliza?

  • Navegación
  • Botones
  • Tarjetas
  • Centramiento
  • Diseño móvil

En proyectos reales suele combinarse con otros conceptos. Conviene observar qué problema resuelve, qué datos necesita y cómo comprobar que el resultado es correcto.

Buenas prácticas

  • Usa gap.
  • Permite wrap.
  • Evita anchos rígidos.
  • Prueba textos largos.
  • Mantén el orden lógico.
Practica con criterio

No memorices reglas aisladas. Comprende el propósito, prueba casos distintos y revisa el resultado antes de usarlo con información importante.

Errores frecuentes

  • Aplicar propiedades al hijo equivocado.
  • Confundir ejes.
  • Provocar desbordamiento.
  • Fijar todos los anchos.
  • Cambiar orden visual sin considerar accesibilidad.

Eje principal, eje transversal y espacio disponible

Flexbox organiza elementos en una dimensión. La propiedad flex-direction define el eje principal; justify-content distribuye espacio en ese eje y align-items alinea en el eje transversal. Comprender estos ejes evita probar propiedades sin saber por qué funcionan.

Los elementos también pueden crecer, encogerse o partir de un tamaño base mediante flex-grow, flex-shrink y flex-basis. La abreviatura flex combina esas decisiones. Para interfaces adaptables suele ser útil permitir salto de línea con flex-wrap y definir un ancho base razonable.

.tarjetas {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}

.tarjeta {
  flex: 1 1 260px;
}

Cada tarjeta intenta partir de 260 píxeles, puede crecer y baja de fila cuando no queda espacio.

Práctica guiada: barra y tarjetas

Crea una barra con logotipo, enlaces y botón. En pantallas amplias deben permanecer en una fila; en pantallas pequeñas, los enlaces pueden ocupar otra línea. Después construye cuatro tarjetas con la misma altura visual aunque tengan distinta cantidad de texto.

  • Prueba justify-content: space-between en la barra.
  • Usa gap en lugar de márgenes individuales.
  • Evita alturas fijas para el texto.
  • Reduce la ventana lentamente y observa el punto exacto donde el diseño deja de funcionar.

Cómo empezar a practicar

  1. Crea una fila.
  2. Centra un bloque.
  3. Activa wrap.
  4. Cambia a columna.
  5. Prueba flex:1.

Conserva cada ejercicio y escribe una breve nota sobre la decisión que tomaste. Después crea una segunda versión más clara o más resistente a errores.

Preguntas frecuentes

¿Sirve para filas y columnas?

Sí, trabaja un eje principal por vez.

¿justify y align?

Actúan en ejes distintos.

¿Cuándo wrap?

Cuando los elementos pueden pasar a otra línea.

¿Reemplaza Grid?

No, se complementan.

¿Qué hace flex:1?

Permite compartir el espacio disponible.
