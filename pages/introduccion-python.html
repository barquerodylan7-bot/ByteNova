Python es un lenguaje de programación conocido por su sintaxis clara y fácil de leer. Es una de las opciones más recomendadas para comenzar a aprender programación.

Con Python puedes crear programas, automatizaciones, aplicaciones web, análisis de datos, herramientas de inteligencia artificial y proyectos de ciberseguridad.

¿Qué es Python?

Python es un lenguaje de programación de propósito general. Esto significa que puede utilizarse para crear muchos tipos diferentes de proyectos.

Su código suele ser más fácil de leer que el de otros lenguajes porque utiliza una estructura limpia y evita símbolos innecesarios.

Ejemplo sencillo 
print("Hola, ByteNova")

Esta instrucción muestra un mensaje en pantalla.

Amplía este tema en: ¿Qué es Python y por qué es ideal para principiantes?

¿Para qué sirve Python?

Automatización

Permite realizar tareas repetitivas de forma automática.

Desarrollo web

Puede utilizarse para crear aplicaciones, servicios y sistemas del lado del servidor.

Análisis de datos

Ayuda a organizar, procesar y analizar grandes cantidades de información.

Inteligencia artificial

Se utiliza en aprendizaje automático, visión artificial y procesamiento de datos.

Ciberseguridad

Permite crear herramientas para analizar redes, registros y procesos.

Educación

Es utilizado para enseñar lógica y fundamentos de programación.

Python es una herramienta, no una solución automática

Para crear proyectos útiles también debes aprender lógica, resolución de problemas y buenas prácticas.

Tu primer programa en Python

Una de las primeras instrucciones que puedes aprender es print(). 

print("Estoy aprendiendo Python")

El texto que está entre comillas aparecerá en la terminal.

Mostrar varios mensajes

print("Hola")
print("Bienvenido a ByteNova")
print("Comencemos a programar")

Mostrar números

print(10)
print(25 + 5)

El segundo ejemplo realiza la suma antes de mostrar el resultado.

Comentarios en Python

Los comentarios permiten escribir explicaciones dentro del código sin que Python las ejecute. 

# Este comentario explica la siguiente línea

print("Hola")

Los comentarios deben utilizarse para aclarar decisiones importantes, no para explicar cada línea evidente.

Variables

Una variable guarda un valor que puede utilizarse dentro del programa. 

nombre = "Ana"
edad = 20

print(nombre)
print(edad)

Nombres recomendados

nombre_usuario = "Carlos"
precio_producto = 1500
usuario_activo = True
Utiliza nombres descriptivos

Un nombre como precio_total es más claro que utilizar solamente x.

Reglas básicas

  • No pueden comenzar con un número.
  • No deben contener espacios.
  • Pueden utilizar guion bajo.
  • Distinguen entre mayúsculas y minúsculas.
  • No deben utilizar palabras reservadas.

Tipos de datos básicos

str

Representa texto. 

nombre = "ByteNova"
int

Representa números enteros. 

edad = 25
float

Representa números decimales. 

precio = 19.99
bool

Representa verdadero o falso. 

activo = True

Consultar el tipo de un dato

edad = 20

print(type(edad))

Convertir datos

numero_texto = "25"
numero = int(numero_texto)

print(numero + 5)
No todos los textos pueden convertirse

Intentar convertir una palabra como "hola" en un número generará un error.

Recibir datos del usuario

La función input() permite recibir texto desde el teclado. 

nombre = input("Escribe tu nombre: ")

print("Hola", nombre)

Recibir un número

El resultado de input() es texto. Para realizar operaciones debes convertirlo. 

edad = int(input("Escribe tu edad: "))

print(edad + 1)
Texto con variables 
nombre = "Laura"
edad = 22

print(f"{nombre} tiene {edad} años")

Operadores

Operadores matemáticos

+

Suma.

-

Resta.

*

Multiplicación.

/

División.

//

División entera.

%

Resto de una división.

numero_uno = 10
numero_dos = 3

print(numero_uno + numero_dos)
print(numero_uno - numero_dos)
print(numero_uno * numero_dos)
print(numero_uno / numero_dos)
print(numero_uno % numero_dos)

Operadores de comparación

10 == 10
10 != 5
10 > 5
10 < 20
10 >= 10
10 <= 15

Estas comparaciones producen un resultado verdadero o falso.

Condiciones

Las condiciones permiten que el programa tome decisiones. 

edad = 18

if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")
La indentación es obligatoria

Python utiliza espacios al inicio de las líneas para identificar qué instrucciones pertenecen a una condición, función o bucle.

Utilizar elif

nota = 85

if nota >= 90:
    print("Excelente")
elif nota >= 70:
    print("Aprobado")
else:
    print("Debes continuar practicando")

Operadores lógicos

edad = 20
tiene_identificacion = True

if edad >= 18 and tiene_identificacion:
    print("Acceso permitido")
  • and: ambas condiciones deben cumplirse.
  • or: al menos una debe cumplirse.
  • not: invierte un valor booleano.

Bucles

Los bucles permiten repetir instrucciones.

Bucle for

for numero in range(5):
    print(numero)

Este ejemplo muestra los números del 0 al 4.

Definir inicio y final

for numero in range(1, 6):
    print(numero)

Bucle while

contador = 1

while contador <= 5:
    print(contador)
    contador += 1
Evita los bucles infinitos

La condición de un bucle while debe cambiar hasta convertirse en falsa.

Listas

Una lista permite guardar varios valores dentro de una sola variable. 

tecnologias = ["HTML", "CSS", "Python"]

print(tecnologias)

Acceder a un elemento

print(tecnologias[0])

Las posiciones comienzan en cero.

Agregar elementos

tecnologias.append("JavaScript")

Eliminar elementos

tecnologias.remove("CSS")

Recorrer una lista

for tecnologia in tecnologias:
    print(tecnologia)

Consultar la cantidad

print(len(tecnologias))

Diccionarios

Un diccionario guarda información mediante pares de claves y valores. 

estudiante = {
    "nombre": "Ana",
    "edad": 20,
    "activo": True
}

print(estudiante["nombre"])

Modificar un valor

estudiante["edad"] = 21

Agregar una propiedad

estudiante["curso"] = "Python"

Recorrer un diccionario

for clave, valor in estudiante.items():
    print(clave, valor)

Funciones

Una función agrupa instrucciones que pueden utilizarse varias veces. 

def saludar():
    print("Hola desde ByteNova")

saludar()

Funciones con parámetros

def saludar(nombre):
    print(f"Hola, {nombre}")

saludar("Carlos")

Devolver un resultado

def sumar(numero_uno, numero_dos):
    resultado = numero_uno + numero_dos
    return resultado

total = sumar(10, 5)

print(total)
Crea funciones pequeñas

Cada función debería tener una responsabilidad clara y fácil de comprender.

Módulos

Un módulo contiene funciones y herramientas que pueden utilizarse dentro de un programa. 

import random

numero = random.randint(1, 10)

print(numero)

Este ejemplo genera un número aleatorio entre 1 y 10.

Módulo math

import math

resultado = math.sqrt(25)

print(resultado)

Importar una función específica

from random import randint

numero = randint(1, 5)

print(numero)

Errores frecuentes

  • Olvidar los dos puntos en una condición.
  • Utilizar una indentación incorrecta.
  • Escribir mal el nombre de una variable.
  • Confundir texto con números.
  • Intentar acceder a una posición inexistente.
  • Crear un bucle que nunca termina.
  • No cerrar comillas o paréntesis.
  • Utilizar una variable antes de crearla.
  • No leer el mensaje de error.
  • Copiar código sin comprenderlo.

SyntaxError

Indica que Python no puede interpretar la forma en que está escrito el código. 

if edad >= 18
    print("Acceso permitido")

Falta colocar los dos puntos después de la condición.

NameError

Puede aparecer cuando se utiliza una variable que no existe. 

print(nombre_usuario)

TypeError

Puede aparecer cuando se intentan combinar tipos de datos incompatibles. 

edad = 20

print("Edad: " + edad)

Una forma correcta sería: 

print(f"Edad: {edad}")
Lee la última línea del error

Generalmente contiene el tipo de problema y una explicación que ayuda a localizarlo.

Proyecto práctico: calculadora sencilla

Este proyecto solicita dos números y una operación. 

numero_uno = float(
    input("Escribe el primer número: ")
)

numero_dos = float(
    input("Escribe el segundo número: ")
)

operacion = input(
    "Escribe +, -, * o /: "
)

if operacion == "+":
    resultado = numero_uno + numero_dos

elif operacion == "-":
    resultado = numero_uno - numero_dos

elif operacion == "*":
    resultado = numero_uno * numero_dos

elif operacion == "/":

    if numero_dos != 0:
        resultado = numero_uno / numero_dos
    else:
        resultado = "No es posible dividir entre cero"

else:
    resultado = "Operación no válida"

print("Resultado:", resultado)

Este proyecto utiliza:

  • Variables.
  • Conversión de datos.
  • Entrada de información.
  • Operadores matemáticos.
  • Condiciones.
  • Validación básica.

Preguntas frecuentes

¿Python es recomendable para principiantes?

Sí. Su sintaxis clara permite concentrarse en aprender lógica y resolución de problemas.

¿Python es gratuito?

Sí. Puede descargarse y utilizarse gratuitamente.

¿Necesito conocimientos previos?

No. Puedes comenzar aprendiendo variables, condiciones, bucles y funciones.

¿Python sirve para inteligencia artificial?

Sí. Es ampliamente utilizado en análisis de datos, aprendizaje automático e inteligencia artificial.

¿Debo memorizar todos los comandos?

No. Lo importante es comprender los conceptos y saber consultar documentación cuando sea necesario.

¿Qué debo aprender después?

Puedes continuar con archivos, manejo de errores, programación orientada a objetos, bases de datos y proyectos prácticos.

Conclusión

Python es un lenguaje flexible, poderoso y accesible para quienes comienzan en programación.

Comprender variables, condiciones, bucles, listas, diccionarios y funciones proporciona una base sólida para desarrollar programas más completos.

Practica con ejercicios pequeños, revisa los errores y mejora cada proyecto de forma progresiva.
