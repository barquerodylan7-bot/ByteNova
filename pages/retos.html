Área 1

Desarrollo web

Practica HTML, CSS y diseño adaptable mediante proyectos sencillos.

Reto 1
🟢 Inicial ⏱️ 30 minutos

Crea tu primera página personal

Construye una página sencilla que presente información sobre ti, tus intereses y tus objetivos.

🎯 Objetivo

Practicar la estructura básica de HTML y las etiquetas de contenido más importantes.

La página debe incluir:

  • Un título principal con tu nombre.
  • Un párrafo de presentación.
  • Una lista con tres intereses.
  • Un subtítulo llamado “Mis objetivos”.
  • Un enlace hacia ByteNova.
  • Un footer con el año actual.
💡 Ver pista

Utiliza las etiquetas <h1>, <p>, <ul>, <a> y <footer>.

✅ Ver solución orientativa

        name="viewport"
        content="width=device-width, initial-scale=1.0"

<body>

        <ul>

            <li>Diseño web</li>

            Quiero crear mis propios proyectos.
        </p>

        <a href="https://bytenova.app/">

        </a>

    </footer>

</html>

Área 2

Programación con Python

Practica variables, entrada de datos, condiciones y operaciones matemáticas.

Reto 3
🟢 Inicial ⏱️ 20 minutos

Crea un saludo personalizado

Construye un programa que solicite el nombre y la edad de una persona.

🎯 Objetivo

Practicar variables, input(), conversión de datos y f-strings.

El programa debe:

  • Solicitar el nombre.
  • Solicitar la edad.
  • Mostrar un saludo personalizado.
  • Indicar cuántos años tendrá el próximo año.
💡 Ver pista

Convierte la edad a número mediante int().

✅ Ver solución orientativa 
nombre = input("Escribe tu nombre: ")

edad = int(
    input("Escribe tu edad: ")
)

proxima_edad = edad + 1

print(
    f"Hola, {nombre}. "
    f"El próximo año tendrás {proxima_edad} años."
)
📘 Repasar Python para principiantes
Reto 4
🟡 Básico ⏱️ 40 minutos

Crea una calculadora sencilla

Crea un programa que permita sumar, restar, multiplicar o dividir dos números.

🎯 Objetivo

Practicar entrada de datos, condiciones y operaciones matemáticas.

El programa debe:

  • Solicitar dos números.
  • Solicitar una operación.
  • Realizar el cálculo correspondiente.
  • Mostrar un mensaje cuando la operación no sea válida.
  • Evitar dividir entre cero.
💡 Ver pista

Utiliza una estructura con if, elif y else.

✅ Ver solución orientativa 
numero_uno = float(
    input("Primer número: ")
)

numero_dos = float(
    input("Segundo número: ")
)

operacion = input(
    "Escribe +, -, * o /: "
)

if operacion == "+":
    resultado = numero_uno + numero_dos
    print(f"Resultado: {resultado}")

elif operacion == "-":
    resultado = numero_uno - numero_dos
    print(f"Resultado: {resultado}")

elif operacion == "*":
    resultado = numero_uno * numero_dos
    print(f"Resultado: {resultado}")

elif operacion == "/":

    if numero_dos == 0:
        print("No se puede dividir entre cero.")
    else:
        resultado = numero_uno / numero_dos
        print(f"Resultado: {resultado}")

else:
    print("La operación no es válida.")

Área 3

Ciberseguridad

Practica cómo reconocer riesgos y mejorar la seguridad de tus cuentas y dispositivos.

Reto 5
🟢 Inicial ⏱️ 15 minutos

Identifica un mensaje de phishing

Analiza el siguiente mensaje ficticio y encuentra al menos cinco señales de advertencia.

De: soporte-banco@seguridad-premios.example

¡URGENTE! Su cuenta será eliminada en 30 minutos. Ingrese inmediatamente al siguiente enlace y confirme su contraseña y código de seguridad.

También ganó un premio especial por ser cliente frecuente.

Enlace: banco-verificacion-premio.example
🎯 Objetivo

Aprender a reconocer urgencia, dominios extraños, solicitudes de credenciales y premios inesperados.

💡 Ver pista

Revisa el remitente, el dominio, el tono del mensaje, la información solicitada y la promesa de un premio.

✅ Ver respuesta orientativa
  • Utiliza una dirección de correo extraña.
  • Crea una sensación extrema de urgencia.
  • Amenaza con eliminar la cuenta.
  • Solicita una contraseña.
  • Solicita un código de seguridad.
  • Ofrece un premio inesperado.
  • El dominio no coincide con el banco.
📘 Repasar qué es el phishing
Reto 6
🟡 Básico ⏱️ 30 minutos

Realiza una revisión de seguridad personal

Revisa tus hábitos digitales utilizando la siguiente lista. No escribas contraseñas ni información privada dentro del ejercicio.

🎯 Objetivo

Detectar medidas de seguridad que todavía puedes mejorar.

Comprueba lo siguiente:

  • Utilizo contraseñas diferentes.
  • Mi correo tiene autenticación en dos factores.
  • Mi teléfono utiliza bloqueo de pantalla.
  • El sistema está actualizado.
  • Tengo una copia de mis archivos importantes.
  • Reviso los enlaces antes de abrirlos.
  • No comparto códigos de verificación.
  • Reviso los permisos de las aplicaciones.
  • No instalo programas de sitios desconocidos.
  • Sé cómo cerrar sesiones abiertas.
💡 Ver recomendación

Selecciona primero dos medidas que todavía no cumples y corrígelas antes de continuar con el resto.

✅ Ver plan de mejora recomendado
  1. Protege primero el correo principal.
  2. Cambia las contraseñas repetidas.
  3. Activa la autenticación en dos factores.
  4. Actualiza tus dispositivos.
  5. Crea una copia de seguridad.
  6. Revisa permisos y sesiones abiertas.
